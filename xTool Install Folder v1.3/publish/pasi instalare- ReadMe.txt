RO:
Se copiaza folderul de instalare pe computer. Se executa fisierul setup.exe. 
Pentru upgradarea la o noua versiune se dezinstaleaza versiunea veche si se reainstaleaza versiunea noua urmand pasul anterior.
Pentru dezinstalare: pt dezinstalare se sterge folderul de instalare si se verifica sa nu ramana instalat in Add or remove programs

EN:
The installation folder is copied to the computer. The setup.exe file is executed.
To upgrade to a new version, uninstall the old version and reinstall the new version following the previous step.
To uninstall: to uninstall, delete the installation folder and check that it does not remain installed in Add or remove programs
